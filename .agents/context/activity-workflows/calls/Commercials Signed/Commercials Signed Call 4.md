# Commercials Signed Call 4 — Onboarding Obstacle Check

## Objective
Identify any internal operational blockers (e.g. key team members on leave) delaying the onboarding kickoff.

## Call Script
"Hi `${Contacts.First_Name}`, `${User.First_Name}` from Jurnii. Hope you're doing well."

*(Wait for response)*

"We received `${Contacts.Account_Name}`'s agreement and want to make sure you start seeing the benefits of our pipeline automation as soon as possible. 

I noticed we haven’t been able to schedule the kickoff meeting yet. Is there an operational blocker or timing issue we should be aware of on our side?"

## Call Outcomes
- **Positive**: Kickoff scheduled.
- **Neutral / No Answer**: Create next call. Send `Commercials Signed Post-Call Email Chain 3`.
- **Negative**: Mark Deferred if delay is requested.
